# **App Name**: Aurum Sanctuary

## Core Features:

- User Authentication: Secure user authentication using Firebase Auth with Google and email passwordless login.
- Journal Entry Creation: Create new journal entries with a rich text editor on the homepage.
- Firestore Integration: Store user data and journal entries in Firestore under 'users' and 'entries' collections.
- Sanctuary Display: Display saved journal entries in a clean, organized manner on the /sanctuary page.
- Entry Sentiment Analysis: Use AI to analyze the sentiment of journal entries and display a mood summary over time. Use AI as a tool that can classify the sentiment.
- Tagging: Allow users to add tags to the entries, to enable categorization, search, and quick filtering.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37) to evoke a sense of luxury and value.
- Background color: Soft beige (#F5F5DC) for a calming and reflective atmosphere.
- Accent color: Muted brown (#8B4513) for secondary elements and highlights.
- Headline font: 'Literata', a serif, for an elegant and readable feel.
- Body font: 'Inter', a sans-serif, to complement the serif headline for a clean, modern feel.
- Use minimalist, line-based icons to maintain a clean and sophisticated aesthetic.
- Maintain a clean and spacious layout with plenty of whitespace to ensure readability and focus.
- Subtle, fade-in animations for entry loading and transitions to enhance user experience.
