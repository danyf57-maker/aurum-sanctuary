# Aurum Positioning Copy Audit

## Goal

Audit the current product copy and information hierarchy against Aurum's target positioning:

1. AI-guided reflection
2. Emotional clarity
3. Private thinking companion
4. Inner pattern discovery

This audit focuses on the highest-leverage surfaces:

- Homepage
- Pricing
- Write
- Journal
- Magazine

It follows the current product structure:

- Write = action
- Journal = memory
- Magazine = reading / analysis

## Summary

The current experience already communicates privacy, softness, and calm extremely well. That strength should be preserved.

The core positioning gap is that Aurum still reads too often as:

- a beautiful private journal
- a stress-relief writing space
- a premium writing ritual

Instead of consistently reading as:

- a private AI-guided reflection system
- a tool for emotional clarity
- a companion for noticing recurring inner patterns over time

In short:

- Privacy is over-expressed
- Calm writing is over-expressed
- AI guidance is under-expressed
- Pattern discovery is under-expressed
- Premium differentiation is too often implied rather than made explicit

## Page-by-page Findings

### Homepage

#### Current issue

The hero and supporting sections strongly express:

- privacy
- emotional softness
- stress release
- calm journaling

But they do not make the product category explicit enough.

Current hero signals:

- "Your mind is full. Lighten it here."
- "Your private journal..."
- "100% encrypted / anonymous / private garden"

This is elegant, but category-risky.

#### What is over-expressed

- private journal language
- emotional unloading
- serenity / calm

#### What is under-expressed

- AI-guided reflection
- reflective prompts and follow-up
- recurring emotional or cognitive patterns
- longitudinal insight
- premium differentiation vs ordinary journaling

#### Recommended direction

Keep the emotional softness, but add one strong explanatory layer:

- Aurum helps you write
- Aurum reflects with you
- Aurum helps recurring patterns become visible

The homepage should make it obvious that the value is not just in writing, but in what Aurum helps reveal over time.

### Pricing

#### Current issue

The pricing page is currently justified mostly through functional features:

- unlimited entries
- unlimited conversations
- history
- export
- reflections

This is too close to category language used by standard journaling products.

#### What is missing

- private-by-design reflection architecture
- AI-guided reflections after writing
- pattern discovery over time
- emotional clarity as the outcome
- explanation of why Aurum sits above standard journaling tools

#### Recommended direction

Keep the pricing layout simple, but shift some copy from feature inventory toward transformation:

- write privately
- receive thoughtful AI reflection
- notice repeated themes and patterns
- build clarity over time

### Write

#### Current issue

Write is correctly positioned as action. That is good.

But the authenticated state still frames the page mainly as a beautiful writing space:

- "Your writing space"
- "Place here what is moving through your mind"

This is emotionally correct, but strategically incomplete.

#### What is missing

- a subtle reminder that Aurum responds with intelligent reflection
- a subtle reminder that writing is the beginning, not the whole experience

#### Recommended direction

Do not overload the page.

Add only a very light layer that makes the sequence clear:

- write privately
- receive reflection
- understand patterns over time

### Journal

#### Current issue

Journal now clearly functions as memory. That is correct and should be preserved.

The remaining gap is that the page mostly behaves like an archive:

- entries
- dates
- excerpts

Even when Aurum reflections are attached, the page does not fully frame why this memory matters.

#### What is missing

- a gentle explanation that revisiting entries helps recurring themes emerge
- a stronger sense that this is memory in service of self-understanding

#### Recommended direction

Do not add heavy modules.

Use one subtle framing sentence or section label to connect memory with pattern discovery.

### Magazine

#### Current issue

Magazine is currently the strongest proof of Aurum's intelligence:

- wellbeing radar
- personality radar
- questionnaire narratives
- dashboard tabs around emotions, themes, Aurum, progression

This is the right zone for pattern discovery.

But the framing is still not explicit enough for positioning purposes.

#### What is missing

- clearer explanation that these are recurring patterns emerging from reflection over time
- clearer answer to: "What am I learning about myself here?"

#### Recommended direction

Magazine should become the clearest expression of:

- AI-guided reflection
- recurring themes
- progression across time
- personal pattern discovery

This can be solved mostly with section labels, helper text, and empty states rather than heavy UI additions.

## What Should Change First

### Priority P0

1. Homepage hero and first explanation layer
2. Pricing transformation copy
3. Magazine framing copy

### Priority P1

1. Write page micro-positioning
2. Journal framing
3. Quiz result modal / first-use expectation

## Copy Principles For Implementation

### Preserve

- softness
- privacy reassurance
- premium calm
- elegant tone

### Strengthen

- guided reflection
- emotional insight
- recurring patterns
- learning over time
- premium distinction

### Avoid

- generic productivity language
- commodity journaling wording
- technical AI jargon
- medical or therapy claims
- busy or overloaded pages

## Acceptance Tests

### Homepage

A first-time visitor should be able to say:

- this is private
- this uses AI to guide reflection
- this helps reveal patterns over time
- this is more than a journal

### Pricing

A reader should understand why Aurum is priced above a standard journaling app.

### Write

A user should understand that writing is the first step in a guided reflection loop.

### Journal

A user should feel that past entries are not just stored, but useful for noticing patterns.

### Magazine

A user should clearly understand what they are learning about themselves and why it matters.
